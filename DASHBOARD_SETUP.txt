════════════════════════════════════════════════════════════
الخطوة 1: إنشاء Google Sheet
════════════════════════════════════════════════════════════
1. افتح: https://sheets.google.com
2. اضغط "Blank" لإنشاء شيت جديد
3. سميه: "Pure Oil Analytics"

════════════════════════════════════════════════════════════
الخطوة 2: فتح Apps Script
════════════════════════════════════════════════════════════
1. من نفس الشيت، اضغط على Extensions (إضافات) فى الـ menu
2. اختار: Apps Script
3. هيفتحلك تبويب جديد فيه محرر كود

════════════════════════════════════════════════════════════
الخطوة 3: لصق الكود
════════════════════════════════════════════════════════════
1. امسح كل الكود الموجود فى محرر Apps Script
2. افتح الملف: APPS_SCRIPT_CODE.txt
3. انسخ كل محتواه والصقه فى المحرر
4. اضغط Ctrl+S لحفظ (سميه: "PureOilBackend")

════════════════════════════════════════════════════════════
الخطوة 4: نشر السكريبت
════════════════════════════════════════════════════════════
1. فى أعلى يمين المحرر، اضغط زرار "Deploy" → "New deployment"
2. اضغط على أيقونة الترس (⚙️) جنب Type → اختار: Web app
3. املأ الحقول:
   • Description: Pure Oil Tracking
   • Execute as: Me
   • Who has access: Anyone
4. اضغط "Deploy"
5. هيطلب أذونات → اضغط "Authorize access" → اختار حسابك →
   "Advanced" → "Go to PureOilBackend (unsafe)" → "Allow"
6. هيظهرلك Web app URL — انسخه!
   شكله كده:
   https://script.google.com/macros/s/AKfycbxxxxxxxxxxxxx/exec

════════════════════════════════════════════════════════════
الخطوة 5: ربط اللينك بالموقع
════════════════════════════════════════════════════════════
ابعتلى اللينك الى نسخته فى الخطوة 4، وانا هاحدثه فى:
1. index.html (الموقع يبدأ يرسل البيانات)
2. dashboard.html (اللوحة تبدأ تقرأ البيانات)
ثم أعمل push على GitHub والـ deploy بيتحدث على Vercel أوتوماتيكياً.
